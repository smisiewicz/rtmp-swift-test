# rtmp-swift-test
Testing RTMP Frameworks


## Setup

Run `setup.sh` to set up Red5 locally and run `./red5.sh` from `red5-server` directory.
This repo uses `Carthage`. Run these steps first:

```
$ cd RTMPStreamingExample
$ brew update
$ brew install carthage
$ carthage update
```
Run Xcode project.
